# Wok，写这个是因为我真破防了

## 起因
我是一个日常会使用三门编程语言的人，这三门分别是：C++（上学用），python（写Agent相关用）， javascript（做开发用）。
但是最开始我先学的C++。嗯，似乎是最麻烦的一个。
然后我学了python。这个很简单。
我花了很多时间用来适应这两门语言之间的切换。好在这两门语言之间的差别还是很大的，目前已经不会有问题了。
问题出在我这次复习JS的时候。Wok，不是，为什么一定要有这样一个“中间值”，夹在python和C++之间啊。Wok，我学的时候就很烦，现在又在FCC上复习一遍。写那种小题目，wok，我要不就是写`num = 4`，要不就是写`int num = 4;`，我实在想不出`let num = 4;`这种设计啊，wok！为什么偏要有这样一个搅屎棍夹在中间啊！
## 焦虑
最近其实挺焦虑的。跟导师来往会有压力，参加一些比赛会有压力，感到灵感枯竭会有压力。不知道是因为到了一个承受压力的年龄，还是因为很多事情发生得太早，让我略有些不知所措。
## 太早的事
好吧，既然提到了这个话题，那似乎就得讲下去了。
写这篇文章的时候我才19岁，但是已经随着一群研究生做了很久的Agent了。似乎按照一个正常的大学生的逻辑，这个时候应该还在钻研C++、做游戏，然后参加一些忽悠人的比赛……好吧，我的人生轨迹似乎走得太快太快了。
![[Pasted image 20260122015815.png]]
这种太快的轨迹给我带来了压力。因为我并不具备相应的基础。我或许有天赋，但这样的天赋只是给了我很多“直观”，给了我很多“正确的感觉”，但给不了我“能力”。
所以我会感觉很难受，会走一步担心一步。
感觉，一瞬间变强的代价就是，心安宁不下来了。
## 喜欢的事业
我很喜欢设计类的工作，致力于实现最佳效率的设计也好，纯粹为了美感的设计也好，我都很喜欢。而Agent和前端正好都是这样的工作。
每次家里有人问起我喜不喜欢我现在的专业时，我都会说：很喜欢，而且，我喜欢的，正好是能赚钱的，还正好是我有天赋的，我很开心。
但是我感觉最近我的喜欢变成了一种愧疚，我开始因为我现在的“半吊子”状态感到愧疚，开始为自己已经走了一段残缺的、不可回头的旅程感到愧疚，开始为我做了很多粗制滥造的Agent，粗制滥造的web感到愧疚。我对我不够牢固的所有的一切，过早发生的所有的一切感到愧疚。
也就是说，我对我喜欢的事业感到十分愧疚。
对粗制滥造感到厌恶，也厌恶以前进行粗制滥造的自己。
