# 番外：风停之后

--written by codex--

## 一
沈屿川到异国的第一周，每天都在调整时差。他最想念的不是食堂的味道，而是深夜电台里那点安静的声音。他把录音笔放在床头，睡前总会听一段林知夏做的节目。那段声音像一条线，把他从陌生城市牵回校园。

他在便利店门口给她发语音：“这边的风很冷，但我听到你的节目就不冷了。”那句话说完，他笑了一下，觉得自己也被谁轻轻抱了一下。

## 二
回国前的最后一晚，他在宿舍楼下录了一段街声，叠上自己的念稿，剪成一条“回家的路”。他把这段音频发给她，说：“等我回来，我们一起把它播出去。”

林知夏收到那段音频时正坐在操场边，她听着风声与他的声音交错，突然明白远行并不是远离，而是把故事拉得更长。她抬头看夜空，心里说：“等你。”
