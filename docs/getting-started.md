# 为什么创造该站点

## 朋友圈发多了
最近朋友圈发得有点太多了，这是一个主要原因。有的时候我并没有那么想分享点啥，但是——如果想写点啥，似乎也只能发在朋友圈了。
放备忘录？总感觉怪怪的；放推特？嘛，我的推特号至今还是对熟人保密的，因为发了很多小隐私、小牢骚。
总之，我不想老在朋友圈骚扰别人，招致被讨厌，又没有别的啥别的方式留住我的思想以及牢骚，就打算整个这个站点了
## 看了innei的网站
我很喜欢一个叫做innei的前端创作者，她写的界面非常漂亮。也正是因为看了她写的界面，我才打算开始了解前端、学习前端（现在我处于一个1/2 Agent builder + 1/3 Frontend engineer + 1/6 Backend engineer的状态）。
这一天，我又打开了[她的个人网站](https://innei.in/) ，就决定自己也整一个了~
哈哈，但是肯定做不了她那么好~

## 洗澡的时候突然有了思考
这几天其实我在做一些课题，洗澡的时候突然对课题有了一个很新颖、很深刻的思考，于是我打算记下来，写好之后发给导师看。
嗯，发个长长的微信信息还是太丑陋了，所以我打算整个这个。
## 突然想起github.io可以白嫖
本来我对于维护一个自己的站点这件事挺纠结的，因为要花钱。
但是我突然想起，开学的时候，班主任发群里的自我介绍用的是一个叫做"XXX.github.io"的站点，就推测它可以白嫖，就了解了一下，果真可以，所以就不犹豫了，直接就是干！
## 想要留下点东西
其实，说实话，最主要的原因还是，希望自己可以在互联网上留下点啥，有种赛博永生的意味。
我很喜欢这种带有**痕迹**或是**记录**，讨厌**错过**或是**遗忘**，总之，我就是想留下点啥。
