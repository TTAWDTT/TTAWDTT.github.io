# 人物小传：薄荷风里的心事

--written by codex--

## 林知夏
来自海边小城的新生，敏感而克制，对声音有近乎直觉的“灵觉”。她从一次紧张的失声里走出来，在电台里重新学会把心放进话筒。她的成长不是突变，而是一步步把恐惧变成习惯，把习惯变成稳定。

## 沈屿川
校电台台长，外冷内热，责任感很强。对他而言，电台是陪伴，也是承诺。他在离开与留下之间摇摆，最终选择把自己的声音留给他想守护的人。

## 许妙妙
林知夏的室友，明亮热烈，像一盏随时会亮起来的灯。她总能用一句玩笑化开紧张，也总在关键时候把人往前推一点。

## 苏晚晴
专业而克制的竞争者。她不擅长表达情绪，却在合作中学会放下防备。她的转变来自承认——优秀并不等于孤独。

## 郭老师
电台的指导老师，话不多却总能一语中的。他像一面镜子，把学生的优点照出来，也把他们该面对的东西轻轻指出来。
