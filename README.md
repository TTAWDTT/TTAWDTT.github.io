# TTAWDTT.github.io

Static GitHub Pages site with a HeroUI-inspired layout and markdown-driven content.

## Quick use
- Edit `content/aboutme.md` for the About page.
- Add new docs in `docs/*.md`.
- Link new docs from `docs/index.md`.

## Local preview
Open `index.html` directly in the browser, or serve the folder with any static server.
